# SailPoint ATM

## Build

- Ensure Java and Maven are installed.
- Build with Maven.
```
mvn install
```

## Test

```
mvn test
```

## Run
- Execute in terminal.

```
java '-cp' 'target\classes' 'com.sailpoint.Atm'
```
- Type a command below.

## Commands

### Log in

```
login 1234
```

### View balance
```
view
```

### Deposit money
```
deposit 100
```

### Withdraw money
```
withdraw 50
```

### Exit
```
exit
```
